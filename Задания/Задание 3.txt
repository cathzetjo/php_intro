I Задание: Нужно создать форму для ввода логина. Файл с формой должен выступать в роли обработчика. После ввода данных нужно проверить пользователя в базе и если пользователь существует то перенаправить его на другую страницу. Если пользователя нет, то под формой вывести сообщение об ошибке.

1) Открыть папку созданную на прошлой практике (в Open Server), либо создать новую
2) Создать файл check.php
3) Вставить разметку html
<!DOCTYPE html>
<html>
 <head>
   <title>Проверка данных</title>
   <meta charset="utf-8">
   <meta name="viewport" content="width=device-width, initial-scale=1.0">
 </head>
 <body>
		
 </body> 
</html>
4) В теге body создать форму проверки логина. Код формы можете посмотреть в учебном приложении, либо скопировать из предыдущей практики
5) В атрибуте action тега Form указать check.php (текущий файл)
6) Добавить файл Db.php в текущий проект, если ранее не добавляли.
7) Подключить его к проекту include "Db.php";
8) Создать файл hello.php. Добавить туда надпись "Добро пожаловать!"
8) Если в массиве $_POST есть данные, то создаем экземпляр класса Db и делаем запрос к базе для проверки существования пользователя с таким логином. 
Если пользователь найден, то делаем перенаправление через функцию header('Location hello.php'). Иначе создаем переменную $error и выводим сообщение под формой на текущей странице
